# v0.1.0 — reproducible Jupiter-signal simulation and recovery

## Summary
- Added `Jupiter_Project_1.ipynb`: a self-contained simulation and analysis notebook demonstrating a Jupiter-like gravitational input embedded in noisy neural population activity and a signal-recovery pipeline.
- Recovered a Jupiter-shaped signal in simulated neurons with Pearson correlation ≈ 0.854 (see notebook Results/plots).
- Added reproducibility and project scaffolding: `README.md`, `requirements.txt`, `LICENSE` (MIT), and `CONTRIBUTING.md`.

## Files added/updated in this release
- `Jupiter_Project_1.ipynb` (primary artifact)
- `README.md` (quickstart + results)
- `requirements.txt` (starter dependency list)
- `LICENSE` (MIT)
- `CONTRIBUTING.md`

## How to reproduce
1. Open the notebook in Colab:
   https://colab.research.google.com/github/victoriamvance-commits/The-Jupiter-project/blob/main/Jupiter_Project_1.ipynb
2. Or run locally:
   git clone https://github.com/victoriamvance-commits/The-Jupiter-project.git
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   jupyter lab
3. Run all notebook cells. Key results (correlation) are produced in the Results section.
