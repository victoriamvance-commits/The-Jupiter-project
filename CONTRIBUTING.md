Contributing
============

Thanks for your interest in contributing to The-Jupiter-project! Below are a few simple ways you can help.

How to contribute
- Open an issue to report bugs, request reproducibility details, or suggest improvements.
- If you want to submit code changes, please open a pull request with a clear description of the change and the motivation.

Code style and testing
- Keep changes small and focused.
- Add tests or a reproducible example when possible.

Communication
- Be respectful and constructive in issue and PR discussions.

Reporting security issues
- For sensitive or security-related issues, please open a private issue or contact the repository owner directly.
